Each backdrop is 1376×768, unmodified, not overwritten, no image generation used on top. The SVG <image> crop box (x=-40 y=-40 w=1250 h=620, xMidYMid slice, onto the 1170×540 viewBox) maps source PNG row → viewBox Y via vbY = row·(1250/1376) + (-40 + (620 - 768·1250/1376)/2) — algebraically identical to round 2's vbY ≈ 0.9084·row − 78.84. For each of the 9 new candidates, a headless Chrome canvas sampled luminance along 21 vertical strips (x = 25%–75% of width, in 2.5% steps, deliberately staying clear of the wide foreground tower trunks) between 36%–80% of image height, found the steepest dark→bright transition per strip (the treeline-base/grass edge — the correct ground-plane vanishing definition, verified visually by overlaying the detected row on 4 sample photos before trusting it for all 9), then took the 5px-bucket mode across strips as the consensus native row. grassTop = yH + (yBottom−yH)·0.05 — a deliberately conservative 5% margin, verified in Node (not just visually) against the actual projection math to sit safely inside the ≤8.465% mathematical ceiling for this PE_A/GROUND_K pair, before this page was ever rendered.
