# Craft Critique — Pass B (Strike Window + Academy)

**Reviewer:** Fable 5 as art director. **Date:** 2026-07-10. **Companion to:** `docs/craft-critique-A.md` (same rubric, same laws — read its §6 before re-scoring).
**Scope:** `geometry-window-mock.html` (landscape instrument, judged at **932×430 and 812×375**) and `academy.html` (portrait-first, judged at **430×932 and 375×812**, plus one 932×430 landscape sanity look, DPR 2, mobile+touch emulation).
**States exercised and screenshotted:** Strike Window — Pure rest (both sizes), tune dock open on Low Point (both sizes, occlusion measured), live slider drag with before-ghost + was→is deltas (session fade extended to inspect), Fat, Duff. Academy — path at 0 XP (top + scrolled), lesson `#/lesson/backspin` (hero, components, spin-loft machine + sliders, myth), quiz wrong-answer + retry + correct-answer, full completion ("mastered" card + badges), path revisited post-completion (completed/available states), 375 portrait spot-check, landscape diagram view. Every px/colour below was **measured in the rendered DOM**.

**Locked, not re-litigated:** P3 palette, C2 Strike Window architecture (window + sequence bar + side column + chip row + dock), Academy 21-lesson structure, portrait-first, dependency-graph unlocks. `docs/strike-window-polish-notes.md` items 1–4 and a11y findings F1–F8 are **adopted elsewhere and not duplicated here** — referenced by ID where my findings extend them. `docs/academy-polish-spec.md` is verdicted in §5.

**Verdict in one line:** both surfaces have the best *content* in the app — an engine-honest instrument and genuinely deep lessons — dressed in the app's worst *token discipline*: a shared-token hijack, three colours of XP, gold spent on locks and warnings, 23 + 15 sub-floor text runs, emoji in a precision instrument, and sticky chrome you can read scrolled content through.

## How to read the punch-lists

Same contract as pass A: atomic items with IDs (`[win-03]`, `[aca-07]`), Fix = do exactly this, Acceptance = what Sonnet checks. **MUST** blocks 90+, **SHOULD** is the 85→92 gap, **COULD** is taste. `→ SYS-n` = fix once in `sa-p3.css` (pass A §5 for SYS-01…10; §3 below for SYS-11…15). Line numbers are current-file; selectors are the stable handles.

## Punch-list

Fix: delete both overrides; decorative borders → `--line` (.10), interactive borders → `--line-strong` (.30), per SYS-07. Acceptance: computed `--line` on `:root` = rgba(255,255,255,.10); no local redeclaration of any sa-p3 token name; every border computes one of the two codified alphas.
