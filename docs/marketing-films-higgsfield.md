# StrikeArc — Higgsfield Film Recipes (Fable 5, 2026-07-10)

**Strategy (locked earlier, executed here):** ONE shared AI shoot → a family of films. Every film lives in the same universe so the feed recognizes the brand in 0.5 s. Film 2 is the straight APP AD; the others feed it.
**Audience law:** the curious golfer, not the expert. Every hook opens with recognizable frustration/curiosity — NEVER telemetry jargon. Real terms appear only *after* the hook, taught in one line.
**Honesty laws:** real UI is always REAL screen capture (never AI-faked UI). No star-ratings shown (under 50 reviews). No fake certainty — the app's own "about 7 in 10" voice is a brand asset. All text INSIDE frame safe-areas.
**Grade:** everything colour-graded to P3 — cold ultraviolet night (#07060C–#140E24 range), ONE warm element per frame (the ember ball/tracer #FF8A4D). If a frame has two warm things, the frame is wrong.
