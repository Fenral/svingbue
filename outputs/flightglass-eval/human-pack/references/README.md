# Referanse-screenshots — tas av eier fra egne enheter

Denne mappen skal fylles med skjermbilder eier tar selv (aldri generert):

1. **Flighty** — hovedskjermen med live flight-tracking (instrument-referansen
   for mono-tall og live sannhet).
2. **Things 3** — en prosjektliste (referansen for ro, hierarki og luft).
3. **Én Trackman-konkurrent** (f.eks. FlightScope/Garmin Golf-appen) — deres
   skjerm for skudd-data (konkurransereferansen Backspin skal slå på lesbarhet).

Ta bildene i samme telefonstørrelse som `../screenshots/` (6.1"–6.7"), lys av,
mørk modus der det finnes. Legg dem her med filnavn `flighty.png`,
`things3.png`, `trackman-konkurrent.png`.

Brukes i parvis sammenligning mot Backspin-flatene før release-beslutning (§6).
